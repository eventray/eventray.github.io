POST /v1/forms/

Create a form.

Request payload

attribute type description
elements array The elements in the form
event_id string (required) The event this form belongs to
name string (required) The name of the form
pages array The pages in this form

Response payload

attribute type description
date_created rfc3339 date/time The date the form was created
date_last_modified rfc3339 date/time The last date/time the form was modified
elements object A map of all the elements in the form
event_id string The event this form belongs to
form_id string The ID of the form
name string The name of the form
theme_id string The ID of the theme applied to this form
user_id string The user that created this form
validation_errors array Array of validation errors preventing the form from being published
