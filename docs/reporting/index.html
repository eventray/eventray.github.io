GET /v1/events/{event_id}/reports/

Get a list of reports belonging to an event. Can be paginated.

Response payload

attribute type description
items array

POST /v1/events/{event_id}/reports/

Creates a new report, which can be run across all attendees in a single event.

Request payload

attribute type description
description string Describe the report
fields Create Report Field array The columns of the report
filter Create Report Field Filter all attendees based on these expressions
group_by Create Report Field Group attendees on this field
name string (required) The name of the report
sort_by Create Report Sort By Sort the report by these fields

Response payload

attribute type description
description string The description of the report
fields Report Field array The columns you want returned in your report
filter Expression Filter your attendees by this expression
group_by Expression Group your attendees on this expression
name string (required) The title of the report
report_id string The id of the report
sort_by Sort Expression array Sort by these expressions

POST /v1/events/{event_id}/reports/one_shot

Run a report without storing it.

Request payload

attribute type description
fields Create Report Field array The columns you want returned in your report
filter Expression Filter your attendees by this expression
group_by Expression Group your attendees on this expression
sort_by Expression array Sort by these expressions

Response payload

attribute type description
filtered integer The number of attendees, after filtering
headers Report Header array Column headers
rows Report Row array Rows of the report
subreport_id string The value indicating which subreport this represents
total integer The number of attendees in the event

GET /v1/events/{event_id}/reports/{report_id}/results

Run a report and get the results.

Response payload

attribute type description
filtered integer The number of attendees, after filtering
headers Report Header array Column headers
report_id string The id of the report
rows Report Row array Rows of the report
subreport_id string The value indicating which subreport this represents
total integer The number of attendees in the event

header string (required) The column header
is_visible boolean UI hint. Defaults to false

attribute type description
descending boolean Reverse the sort order. Defaults to false
expression Expression (required) The expression to sort on
