POST /v1/sessions/password

Create a session. All calls to the API need an authorization header and a token. After creating a session, pass the token to the API as a header: Authorization: token: {token}

Request payload

attribute type description
email_address string Email address of the user
password string The user's password

Response payload

attribute type description
email_address string Email address of the user
token string The user's token.
