Operations

Binary expression

Operations that take two values

attribute description
op The name of the operation
left, right The values for the operation

Binary operations

name description
equals left == right
not equals left != right
gt left > right
ge left >= right
lt left < right
le left <= right
in left in right
not in left not in right
has all of left has all of the items in right
has one of left has one of the items in right
has only left has exactly the same items as right
matches left is matched by the regex in right
string_contains left contains the text in right
string_starts_with left starts with the text in right
string_ends_with left ends with the text in right

Variadic expression

attribute description
op The name of the operation
operands The values to be operated upon

Variadic operations

name description
and All expressions values must be true
or One of the expressions must be true

Unary expression

attribute description
op The name of the operation
operand The value to be operated upon

name description
